Compte sécurisé d’IRCC: Se connecter

Pour les demandes prises en charge, vous pouvez utiliser ce compte pour :

  • commencer une demande;
  • présenter votre demande et payer les frais;
  • recevoir des messages liés à votre demande;
  • vérifier l’état de votre demande;
  • mettre à jour vos renseignements.

Certains types de demande ne sont pas accessibles dans le compte sécurisé. Pour savoir si vous pouvez utiliser ce compte, vérifiez les instructions relatives à votre demande.

Compte sécurisé d’IRCC

Ouvrez une session en utilisant:

  • Votre nom d’utilisateur et mot de passe CléGC GCKeysign-in
  • Votre partenaire de connexion Interac® Canadien banque
Vous ne savez pas comment ouvrir une session?

Ouvrez une session de la même manière chaque fois que vous utilisez votre compte :

  • lorsque vous vous êtes inscrit pour la première fois à ce compte, vous avez choisi une option d’ouverture de session telle qu’un nom d’utilisateur et un mot de passe CléGC, votre partenaire de connexion bancaire ou votre province
  • l’option d’ouverture de session que vous avez choisie est liée à votre compte et à votre profil d’utilisateur – si vous en choisissez une autre, il se peut que l’on vous demande de vous réinscrire

Si vous ne vous êtes pas déjà inscrit à un compte, inscrivez-vous maintenant.

Si vous avez oublié votre nom d’utilisateur ou mot de passe

Choisissez les instructions correspondant à votre situation :

Vous avez oublié votre nom d’utilisateur ET votre mot de passe

Si vous avez ajouté une adresse courriel de récupération à CléGC :

  1. Sélectionnez votre nom d’utilisateur et mot de passe CléGC
  2. Sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC pour qu’il vous soit envoyé par courriel
  3. Une fois que vous avez reçu votre nom d’utilisateur par courriel, sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC et suivez les étapes pour le réinitialiser
Si vous n’avez pas ajouté d’adresse courriel de récupération à CléGC :
  1. Sélectionnez Votre nom d’utilisateur et mot de passe CléGC
  2. Sélectionnez le bouton s'inscrire sur la page Bienvenue au service CléGC pour créer un nouveau nom d'utilisateur et un nouveau mot de passe CléGC et ouvrir un nouveau compte
  3. Au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte
Vous avez oublié votre nom d’utilisateur
  1. Sélectionnez Votre nom d’utilisateur et mot de passe CléGC
  2. Sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC
  3. Saisissez votre courriel de récupération sur la page Récupérer le nom d’utilisateur - Étape 1 sur 2 :
    • vous avez fourni votre courriel de récupération soit lorsque vous vous êtes inscrit pour obtenir votre nom d’utilisateur et votre mot de passe de CléGC, soit à un moment ultérieur
    • si vous n'avez pas ajouté d'adresse électronique de récupération et que vous avez oublié votre nom d'utilisateur et votre mot de passe, sélectionnez le bouton s'inscrire sur la page Bienvenue au service CléGC pour créer un nouveau nom d'utilisateur et un nouveau mot de passe CléGC, et enregistrez-vous à nouveau
  4. Sélectionnez Continuer
    • Vous serez amené à une page qui explique où votre nom d’utilisateur sera envoyé par courriel et ce qu’il faut faire si vous ne recevez pas le courriel.
  5. Sélectionnez encore une fois Continuer
  6. Vérifiez vos courriels. Vous devriez recevoir un courriel du gouvernement du Canada comportant votre nom d’utilisateur de CléGC
  7. Saisissez votre nom d’utilisateur et votre mot de passe sur la page Bienvenue au service CléGC
Vous avez oublié votre mot de passe
  1. Sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC
  2. Saisissez votre nom d’utilisateur de CléGC
  3. Sélectionnez Continuer
    • Vous serez amené à une page qui explique où votre nom d’utilisateur sera envoyé par courriel et ce qu’il faut faire si vous ne recevez pas le courriel
    • Si vous n’avez pas saisi d’adresse courriel lors de votre inscription, suivez les instructions dans la case Questions et réponses de récupération
Réinitialisation du mot de passe à l’aide de l’adresse courriel
  1. Si vous avez saisi votre adresse courriel, vérifiez vos courriels. Vous devriez recevoir un courriel du gouvernement du Canada comportant un code de confirmation
  2. Saisissez le code de confirmation que vous avez reçu sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4 (cette page devrait s’être chargée automatiquement après que vous avez saisi votre nom d’utilisateur et sélectionné Continuer sur la page Vous avez oublié votre mot de passe? - Étape 1).
  3. Sélectionnez Continuer
  4. Vous serez invité à répondre à une question de récupération
  5. Saisissez la réponse à la question dans le champ Ma réponse de récupération
  6. Sélectionnez Continuer
  7. Saisissez un nouveau mot de passe dans le champ Nouveau mot de passe
  8. Saisissez de nouveau ce même mot de passe dans le champ Confirmer le nouveau mot de passe
  9. Sélectionnez Continuer
  10. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe? - Complété
Réinitialisation du mot de passe à l’aide des questions de récupération
  1. Sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4, sélectionnez le bouton Questions et réponses de récupération dans la case Questions et réponses de récupération
  2. Saisissez vos réponses dans les champs Ma réponse de récupération, Ma personne mémorable et Ma date mémorable (les questions et indices devraient s’afficher au-dessus de chaque champ)
  3. Saisissez un nouveau mot de passe dans le champ Créez votre nouveau mot de passe
  4. Saisissez le même mot de passe dans le champ Confirmez votre nouveau mot de passe
  5. Sélectionnez Continuer
  6. Si vous avez répondu correctement à toutes les questions de récupération, vous serez dirigé vers une page intitulée Vous avez oublié votre mot de passe? - Complété
  7. Sélectionnez Continuer
Changer ou utiliser l'authentification à deux facteurs de CléGC

L'utilisation de l'authentification à deux facteurs à chaque fois que vous vous connectez protège votre compte. Vous devrez choisir une deuxième méthode d'authentification en plus de votre nom d'utilisateur et de votre mot de passe.

Configuration de votre méthode d'authentification à deux facteurs

Changer l'authentification à deux facteurs

Vous devez avoir accès à votre compte de messagerie existant ou à l'application d'authentification, ou encore aux codes de récupération que vous avez enregistrés :

si vous avez encore accès à votre méthode à deux facteurs

Sélectionnez Gérer votre authentification à deux facteurs sur la page d'accueil affichée après avoir saisi votre nom d'utilisateur et votre mot de passe CléGC

  • puis sélectionnez le bouton Modifier votre authentification à deux facteurs sur la page Succès de l'authentification
si vous n'avez pas accès à votre méthode d'authentification à deux facteurs

Utiliser un code de récupération sauvegardé :

  • sélectionner le bouton Récupérer le compte sur la page de réussite de l'authentification
  • saisissez l'un des codes de récupération que vous avez enregistrés lors de la configuration initiale de l'authentification à deux facteurs
  • sélectionnez ensuite le bouton Modifier votre authentification à deux facteurs sur la page Succès de l'authentification.

Aucun code de récupération :

Une liste de codes de récupération à enregistrer vous a été fournie lorsque vous avez sélectionné votre méthode d'authentification à deux facteurs. Si vous ne les avez pas :

  • Créez un nouveau compte et inscrivez-vous pour obtenir un nouveau nom d'utilisateur et un nouveau mot de passe CléGC
  • Enregistrez votre nouvelle série de codes de récupération lorsque vous ajoutez l'authentification à deux facteurs
  • après votre inscription, vous pourrez peut-être récupérer votre compte ou lier des applications existantes à votre nouveau compte

CléGC a été révoquée

Si votre CléGC a été révoquée, vous ne pouvez pas la récupérer.

Votre nom d'utilisateur et votre mot de passe GCKey peuvent être révoqués s'ils sont compromis ou s'ils n'ont pas été utilisés au moins une fois au cours des deux dernières années. Vous devrez :

  • s'inscrire à nouveau pour le service
  • vous pourrez peut-être utiliser une option de connexion différente ou sur la page Bienvenue au service CléGC,sélectionnez le bouton S’enregistrer et créez un nouveau nom d’utilisateur et un nouveau mot de passe de CléGC
  • Au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte

Votre CléGC peut être révoqué si votre nom d'utilisateur et votre mot de passe ont été compromis. N'utilisez pas le même nom d'utilisateur et le même mot de passe que pour CléGC pour d'autres comptes en ligne.

Messages d'erreur et problèmes techniques

Vous pouvez utiliser le même nom d’utilisateur et le même mot de passe de CléGC sur différents ordinateurs, téléphones et tablettes.

Si vous obtenez des messages d’erreur :

  • utilisez un navigateur Web moderne comme Safari, Chrome, Firefox or Microsoft Edge, plutôt qu’Internet Explorer
  • n’ajoutez pas à vos favoris les pages que vous voyez après vous être connecté – vous devez vous connecter pour les voir maintenez votre navigateur à jour
  • utilisez les paramètres par défaut de votre navigateur, car CléGC nécessite l’activation de JavaScript et des témoins

Si vous voyez l’un de ces messages lorsque vous essayez de vous connecter, mettez votre navigateur à jour :

  • Impossible d’afficher la page
  • Impossible d’établir une connexion sécurisée
  • Page non disponible

Vous trouverez généralement l’option de mise à jour de votre navigateur dans les paramètres ou les préférences du menu de votre navigateur. Si vous utilisez Internet Explorer, mettez à jour votre navigateur en passant à Microsoft Edge.

Autres problèmes techniques
  • Trop de tentatives :si vous faites trop de tentatives infructueuses pour vous connecter au cours de la même session, un message d'échec peut s'afficher. Fermez la fenêtre ou l'onglet de votre navigateur et recommencez.
  • Difficultés techniques : Le site pourrait être indisponible pour un certain nombre de raisons, y compris une brève période de maintenance du système. Nous nous excusons pour la gêne occasionnée. Réessayez plus tard.
  • Session expirée pour des raisons de sécurité, votre session sera fermée si elle est inactive pendant un certain nombre de minutes. Reconnectez-vous.
