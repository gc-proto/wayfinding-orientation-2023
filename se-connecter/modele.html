[facultatif] Comment obtenir un nouveau code

Vous avez oublié votre mot de passe ou nom d’utilisateur CléGC

Vous avez oublié votre mot de passe ou nom d’utilisateur CléGC

Choisissez les instructions qui s’appliquent à votre situation :

Vous avez oublié votre nom d’utilisateur ET votre mot de passe

Si vous avez ajouté une adresse courriel de récupération à CléGC :

  1. Sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC pour qu’il vous soit envoyé par courriel
  2. Une fois que vous avez reçu votre nom d’utilisateur par courriel, sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC et suivez les étapes pour le réinitialiser
Si vous n’avez pas ajouté d’adresse courriel de récupération à CléGC
  1. Sélectionner le bouton S’enregistrer sur la page d’accueil Bienvenue au service CléGC afin de créer un nouveau nom d’utilisateur et un nouveau mot de passe CléGC et de vous inscrire à un nouveau compte
  2. Au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte
Vous avez oublié votre nom d’utilisateur
  1. Sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC
  2. Saisissez votre courriel de récupération sur la page Récupérer le nom d’utilisateur - Étape 1 sur 2 :/strong>
    • vous avez fourni votre courriel de récupération soit lorsque vous vous êtes inscrit pour obtenir votre nom d’utilisateur et votre mot de passe de CléGC, soit à un moment ultérieur
    • si vous n’avez pas ajouté d’adresse courriel de récupération et que vous avez oublié votre nom d’utilisateur et votre mot de passe, vous devez vous inscrire pour obtenir une nouvelle CléGC et vous réinscrire pour chacun des comptes que vous utiliseze
  3. Sélectionnez Continuer
    • Vous serez amené à une page qui explique où votre nom d’utilisateur sera envoyé par courriel et ce qu’il faut faire si vous ne recevez pas le courriel.
  4. Sélectionnez encore une fois Continuer
  5. Vérifiez vos courriels. Vous devriez recevoir un courriel du gouvernement du Canada comportant votre nom d’utilisateur de CléGC
  6. Ouvrez une session à l’aide de votre nom d’utilisateur et de votre mot de passe sur la page Bienvenue au service CléGC
Vous avez oublié votre mot de passe
  1. Sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC
  2. Saisissez votre nom d’utilisateur de CléGC
  3. Sélectionnez Continuer
    • Vous serez amené à une page qui explique où votre nom d’utilisateur sera envoyé par courriel et ce qu’il faut faire si vous ne recevez pas le courriel.
    • Si vous n’avez pas saisi d’adresse courriel lors de votre inscription, suivez les instructions dans la case Questions et réponses de récupération
Réinitialisation du mot de passe à l’aide de l’adresse courriel
  1. Si vous avez saisi votre adresse courriel, vérifiez vos courriels. Vous devriez recevoir un courriel du gouvernement du Canada comportant un code de confirmation
  2. Saisissez le code de confirmation que vous avez reçu sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4 (cette page devrait s’être chargée automatiquement après que vous avez saisi votre nom d’utilisateur et sélectionné Continuer sur la page Vous avez oublié votre mot de passe? - Étape 1).
  3. Sélectionnez Continuer
  4. Vous serez invité à répondre à une question de récupération
    • Il s’agit d’une question à laquelle vous avez répondu lorsque vous avez créé votre nom d’utilisateur et votre mot de passe de CléGC
  5. Saisissez la réponse à la question dans le champ Ma réponse de récupération
  6. Sélectionnez Continuer
  7. Saisissez un nouveau mot de passe dans le champ Nouveau mot de passe
  8. Saisissez de nouveau ce même mot de passe dans le champ Confirmer le nouveau mot de passe
  9. Sélectionnez Continuer
  10. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe? - Complété
Réinitialisation du mot de passe à l’aide des questions de récupération
  1. Sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4, sélectionnez le bouton Questions et réponses de récupération dans la case Questions et réponses de récupération
  2. Saisissez vos réponses dans les champs Ma réponse de récupération, Ma personne mémorable et Ma date mémorable (les questions et indices devraient s’afficher au-dessus de chaque champ)
  3. Saisissez un nouveau mot de passe dans le champ Créez votre nouveau mot de passe
  4. Saisissez le même mot de passe dans le champ Confirmez votre nouveau mot de passe
  5. Sélectionnez Continuer
  6. Si vous avez répondu correctement à toutes les questions de récupération, vous serez dirigé vers une page intitulée Vous avez oublié votre mot de passe? - Complété
  7. Sélectionnez Continuer
Nom d’utilisateur et mot de passe CléGC révoqués

Votre nom d’utilisateur et votre mot de passe CléGC ont été révoqués ou sont manquants, invalides ou incomplets

Si vous n’avez pas ouvert une session avec votre nom d’utilisateur et votre mot de passe CléGC au moins une fois au cours des deux dernières années, ils ont été révoqués et supprimés. Vous devrez :

  • vous réinscrire au service again
  • au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte

Changez de partenaire de connexion

Utilisez un autre partenaire de connexion

Si vous avez changé de banque ou de coopérative de crédit et que vous avez déjà ouvert une session par l’entremise d’un partenaire de connexion Interac®, vous pouvez être en mesure de changer de partenaire :

  • sélectionnez Changer mon partenaire de connexion dans la barre de menu de la page Partenaire de connexion Interacpage

Messages d’erreur du navigateur

Votre navigateur montre un message d’erreur

Mettez votre navigateur à jour si vous voyez l’un de ces messages lorsque vous essayez de vous connecter :

  • Impossible d’afficher la page
  • Impossible d’établir une connexion sécurisée
  • Page non disponible

Vous trouverez généralement l’option de mise à jour de votre navigateur dans les paramètres ou les préférences du menu de votre navigateur. Les dernières versions des navigateurs sont dotées de fonctions de sécurité pour protéger vos renseignements personnels.

Utilisez un navigateur Web moderne comme Google Chrome, Safari ou Microsoft Edge, plutôt qu’Internet Explorer.
