Aide avec votre compte

Vous avez oublié votre mot de passe ou nom d’utilisateur de CléGC

Veuillez lire les instructions correspondant à votre situation.

Si vous avez oublié votre nom d’utilisateur et votre mot de passe
  1. Suivez d’abord les étapes pour récupérer votre nom d’utilisateur.
  2. Une fois que vous avez récupéré votre nom d’utilisateur, suivez les étapes pour récupérer votre mot de passe.
Vous avez oublié votre nom d’utilisateur
Si vous avez une adresse courriel de récupération :
  1. sélectionnez Nom d’utilisateur et mot de passe de CléGC;
  2. sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC;
  3. saisissez votre adresse courriel de récupération sur la page Récupérer le nom d’utilisateur – Étape 1 sur 2;
    • vous avez fourni votre courriel de récupération lorsque vous vous êtes inscrit à CléGC, ou à un moment ultérieur;
  4. sélectionnez Continuer;
    • vous serez dirigé vers une page qui explique où votre nom d’utilisateur sera envoyé par courriel et ce qu’il faut faire si vous ne recevez pas le courriel;
  5. sélectionnez Continuer;
  6. vérifiez vos courriels;
    • vous devriez recevoir un courriel du CléGC contenant votre nom d’utilisateur de CléGC;
  7. connectez‑vous avec votre nom d’utilisateur et votre mot de passe sur la page Bienvenue au service CléGC.
Si vous n’avez pas d’adresse courriel de récupération :
  1. sélectionnez Nom d’utilisateur et mot de passe de CléGC;
  2. sélectionnez Se connecter sur la page Bienvenue au service CléGC pour créer un nouveau nom d’utilisateur et un nouveau mot de passe de CléGC;
  3. créez un nouveau compte dans le compte sécurisé d’IRCC.

Pendant ce processus, vous pourrez peut‑être  récupérer votre compte

Vous avez oublié votre mot de passe
  1. Sélectionnez Nom d’utilisateur et mot de passe de CléGC.
  2. Sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC.
  3. Saisissez votre nom d’utilisateur de CléGC.
  4. Sélectionnez Continuer.
    • Si vous avez saisi une adresse courriel pour obtenir votre nom d’utilisateur de CléGC, réinitialisez votre mot de passe avec votre adresse courriel.
    • Si vous n’avez pas saisi d’adresse courriel lors de votre inscription, réinitialisez votre mot de passe à l’aide des questions de récupération.
Réinitialisation du mot de passe à l’aide de l’adresse courriel
  1. Saisissez votre adresse courriel.
  2. Vérifiez vos courriels. Vous devriez recevoir un courriel du CléGC contenant un code de confirmation.
  3. Saisissez le code de confirmation sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4.
    • Cette page devrait s’être chargée automatiquement après que vous avez saisi votre nom d’utilisateur et sélectionné Continuer sur la page précédente.
  4. Sélectionnez Continuer.
  5. Vous serez invité à répondre à une question de récupération. Saisissez la réponse à la question dans le champ Ma réponse de récupération.
  6. Sélectionnez Continuer.
  7. Saisissez un nouveau mot de passe dans le champ Nouveau mot de passe.
  8. Saisissez de nouveau le mot de passe dans le champ Confirmer le nouveau mot de passe.
  9. Sélectionnez Continuer.
  10. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe – Complété.
Réinitialisation du mot de passe à l’aide des questions de récupération
  1. Sur la page Vous avez oublié votre mot de passe? – Étape 2 sur 4, sélectionnez Questions et réponses de récupération dans la case Questions et réponses de récupération.
  2. Saisissez vos réponses dans les champs Ma réponse de récupération, Ma personne mémorable et Ma date mémorable.
    • Les questions et indices devraient s’afficher au‑dessus de chaque champ.
  3. Saisissez un nouveau mot de passe dans le champ Créez votre nouveau mot de passe.
  4. Saisissez le même mot de passe dans le champ Confirmez votre nouveau mot de passe.
  5. Sélectionnez Continuer.
  6. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe? - Complété
    • Si vous avez répondu correctement à toutes les questions de récupération, vous serez automatiquement dirigé vers cette page.
Changer ou utiliser l'authentification à deux facteurs de CléGC

L'utilisation de l'authentification à deux facteurs à chaque fois que vous vous connectez protège votre compte. Vous devrez choisir une deuxième méthode d'authentification en plus de votre nom d'utilisateur et de votre mot de passe.

Configuration de votre méthode d'authentification à deux facteurs

Changer l'authentification à deux facteurs

Vous devez avoir accès à votre compte de messagerie existant ou à l'application d'authentification, ou encore aux codes de récupération que vous avez enregistrés :

Si vous avez toujours accès à votre méthode à deux facteurs

Sélectionnez Gérer votre authentification à deux facteurs sur la page d'accueil qui s'affiche après avoir saisi votre nom d'utilisateur et votre mot de passe CléGC

  • sélectionnez ensuite le bouton Modifier votre authentification à deux facteurs sur la page Succès de l'authentification
Si vous n'avez pas accès à votre méthode d'authentification à deux facteurs

Utiliser un code de récupération sauvegardé :

  • sélectionner le bouton Récupérer le compte sur la page Authentification à deux facteurs
  • saisissez l'un des codes de récupération que vous avez enregistrés lors de la configuration initiale de l'authentification à deux facteurs
  • sélectionnez ensuite le bouton Modifier votre authentification à deux facteurs sur la page Succès de l'authentification.

Aucun code de récupération :

Une liste de codes de récupération à enregistrer vous a été fournie lorsque vous avez sélectionné votre méthode d'authentification à deux facteurs. Si vous ne les avez pas :

  • Créez un nouveau compte et inscrivez-vous pour obtenir un nouveau nom d'utilisateur et un nouveau mot de passe CléGC
  • Enregistrez votre nouvelle série de codes de récupération lorsque vous ajoutez l'authentification à deux facteurs
  • Au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte

CléGC a été révoquée

Si votre nom d’utilisateur et mot de passe CléGC ont été révoquée, vous ne pouvez pas les récupérer.

Ils peuvent avoir été compromis ou n'ont pas été utilisés au moins une fois au cours des deux dernières années. Vous devrez :

  • s'inscrire à nouveau pour le service
  • vous pourrez utiliser une option de connexion différente ou sélectionner le bouton S’enregistrer sur la page Bienvenue au service CléGC pour créer un nouveau nom d’utilisateur et un mot de passe CléGC
  • Au cours de ce processus, vous serez peut-être en mesure de récupérer votre compte ou de relier des demandes existantes à votre nouveau compte

Votre CléGC peut être révoquée si votre nom d'utilisateur et votre mot de passe ont été compromis. N'utilisez pas le même nom d'utilisateur et mot de passe que vous avez enregistré pour CléGC ou d'autres comptes en ligne.

Messages d'erreur et problèmes techniques

Vous pouvez utiliser le même nom d’utilisateur et le même mot de passe de CléGC sur différents ordinateurs, téléphones et tablettes.

Si vous obtenez des messages d’erreur :

  • utilisez un navigateur Web moderne comme Safari, Chrome, Firefox or Microsoft Edge, plutôt qu’Internet Explorer
  • n’ajoutez pas à vos favoris les pages que vous voyez après vous être connecté – vous devez vous connecter pour les voir maintenez votre navigateur à jour
  • utilisez les paramètres par défaut de votre navigateur, car CléGC nécessite l’activation de JavaScript et des témoins

Si vous voyez l’un de ces messages lorsque vous essayez de vous connecter, mettez votre navigateur à jour :

  • Impossible d’afficher la page
  • Impossible d’établir une connexion sécurisée
  • Page non disponible

Vous trouverez généralement l’option de mise à jour de votre navigateur dans les paramètres ou les préférences du menu de votre navigateur. Si vous utilisez Internet Explorer, mettez à jour votre navigateur en passant à Microsoft Edge.

Autres problèmes techniques
  • Trop de tentatives :si vous faites trop de tentatives infructueuses pour vous connecter au cours de la même session, un message d'échec peut s'afficher. Fermez la fenêtre ou l'onglet de votre navigateur et recommencez.
  • Difficultés techniques : le site pourrait être indisponible pour un certain nombre de raisons, y compris une brève période de maintenance du système. Nous nous excusons pour la gêne occasionnée. Réessayez plus tard.
  • Session expirée :pour des raisons de sécurité, votre session sera fermée si elle est inactive pendant un certain nombre de minutes. Reconnectez-vous.
